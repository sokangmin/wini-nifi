# Apache MiNiFi

MiNiFi는 Apache NiFi의 서브 프로젝트로써, NiFi의 데이터 흐름관리 기본기능을 위주로 생성소스의 데이터수집에 중점을 둔 작고 가벼운 agent개념의 솔루션이다.<br/>
NiFi는 주로 데이터센터안의 엔터프라이즈급 서버나 클러스터에 설치되며. MiNiFi는 감지기 또는 생성소스와 가까운 IoT용 저전력기기에 설치된다.<p/>
